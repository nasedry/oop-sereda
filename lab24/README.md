# Лабораторна робота №24  
## Strategy + Observer

### Що реалізовано

У цій роботі я використала два патерни:

**Strategy**  
Створено інтерфейс `INumericOperationStrategy` і три реалізації:
- квадрат числа
- куб числа
- квадратний корінь  

Клас `NumericProcessor` дозволяє змінювати стратегію під час виконання програми.

**Observer**  
Створено клас `ResultPublisher` з подією `ResultCalculated`.  
Є три спостерігачі:
- вивід у консоль
- збереження історії
- повідомлення, якщо результат більший за поріг  

Після кожного обчислення всі підписані спостерігачі автоматично отримують результат.

### Висновок

Я навчилась використовувати Strategy для зміни алгоритмів під час роботи програми та Observer для автоматичного сповіщення інших класів.
