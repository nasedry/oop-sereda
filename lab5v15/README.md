## Контрольні запитання та відповіді

Що таке generics і які їх переваги
Generics — це типи, які працюють із будь-якими даними. Вони дозволяють писати універсальний код без копіювання для кожного типу і роблять його безпечнішим для типів.
Які основні відмінності між ArrayList і List<T>
ArrayList може зберігати будь-які об’єкти, але не перевіряє тип під час компіляції. List<T> типізований, безпечний для типів і працює швидше.
Чим відрізняється Dictionary<TKey,TValue> від List<T>
List<T> — це просто список елементів, шукає по індексу. Dictionary<TKey,TValue> зберігає пари ключ-значення і швидко шукає по ключу.
У чому перевага LINQ над класичними циклами
LINQ дозволяє писати компактний і легший для читання код при роботі з колекціями: фільтрувати, сортувати і групувати без ручних циклів.
Як працює ключове слово finally
finally виконується завжди після try-catch, навіть якщо був виняток. Зручно для закриття файлів, з’єднань і очищення ресурсів.
Коли доцільно створювати власні класи винятків
Коли хочеш чітко сказати, яка саме помилка сталася і обробити її окремо. Наприклад тут створено InvalidShipmentDatesException, щоб відловлювати помилки дат відправлення.
