# Лабораторна 7 - Контрольні питання

## 1. Які типи винятків найчастіше виникають при роботі з файлами та мережею?
- При роботі з файлами можуть виникати помилки вводу/виводу (`IOException`) або коли файл не знайдено (`FileNotFoundException`).  
- При роботі з мережею можуть виникати помилки під час запиту (`HttpRequestException`) або проблеми з підключенням (`SocketException`).

## 2. Поясніть принцип роботи патерну Retry. Коли його доцільно використовувати?
- Retry пробує виконати операцію кілька разів, якщо вона спочатку не вдалась.  
- Його застосовують, коли помилка тимчасова, наприклад, мережевий збій або файл тимчасово заблокований.

## 3. Як реалізувати експоненційну затримку між повторними спробами?
- Починаємо з невеликої затримки, наприклад 1 секунда.  
- Після кожної невдалої спроби збільшуємо час очікування в 2 рази, щоб не перевантажувати систему.

## 4. Для чого потрібен делегат shouldRetry у RetryHelper?
- Він визначає, для яких помилок робити повторну спробу, а для яких – ні.  
- Це допомагає уникнути повторів для помилок, які точно не виправляться повтором.
